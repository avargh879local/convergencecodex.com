# SEO Growth Plan

Last updated: 2026-05-03

## What was added

- Topic pages for long-tail search queries.
- Unique title and meta description on the homepage and each topic page.
- Canonical URLs pointing to `https://www.convergencecodex.com/`.
- A fuller `sitemap.xml`.
- About, Methodology, Privacy, Contact, and Sources pages for trust.
- Internal links from the homepage to every topic page.
- Advanced structured data, visible FAQ blocks, RSS feed, social preview card, and `llms.txt`.

## First launch tasks after deployment

1. Confirm the site is public in an incognito browser.
   - `https://www.convergencecodex.com/`
   - `https://www.convergencecodex.com/sitemap.xml`

2. Add the site to Google Search Console.
   - Use the Domain property if possible.
   - Submit `https://www.convergencecodex.com/sitemap.xml`.
   - Request indexing for the homepage and `/mahdi-vs-antichrist/`.
   - Inspect `/mahdi-vs-antichrist/`, `/antichrist-in-the-bible/`, and `/who-is-dajjal/` after deployment.

3. Add the site to Bing Webmaster Tools.
   - Import from Google Search Console if available.
   - Submit the same sitemap.

4. Create a public contact mailbox before applying for AdSense.
   - Recommended: `research@convergencecodex.com`
   - Cloudflare Email Routing can forward it to a private inbox.

5. Do not add ads until the site has:
   - A working contact channel.
   - At least 15 to 25 substantial public pages.
   - No Cloudflare Access login on public pages.
   - No hate, harassment, or derogatory language toward any religious group.

## Promotion without much money

- Post one short YouTube/TikTok/Reels clip per topic page.
- Write careful answers on Quora and link only when directly relevant.
- Share source tables in Reddit discussions without spamming.
- Create X/Twitter threads from one comparison table at a time.
- Ask scholars or prophecy channels for correction, not promotion.

## Paid tests

If spending money, keep it tiny:

- Start with $5 to $10 per day for 3 to 5 days.
- Promote only one beginner page, not the homepage.
- Best test page: `/mahdi-vs-antichrist/`.
- Stop the test if visitors bounce immediately or do not read.

Never buy backlinks. Google explicitly warns that paid links can violate spam policies.
